Projects and Learning Experience

This page highlights a selection of my academic work, technical projects, and professional learning experiences that reflect my growth in programming, problem-solving, and collaboration.

Java - Tic Tac Toe

This was a final group project for my Java course where we developed a Tic Tac Toe game. My contributions included adding different game modes, implementing score tracking, and making sure the game reset properly after each round. This project helped strengthen my Java programming skills, teamwork, and problem-solving in a collaborative environment.

Tools used: Java, IntelliJ IDEA, Discord

Supervised Machine Learning

This course introduced me to the fundamentals of machine learning, including concepts such as linear regression and model training. Through hands-on exercises, I gained experience applying Python to beginner-level machine learning tasks and developed a stronger understanding of how data can be used to build predictive models.

Tools used: Python, Coursera

Google AI Essentials

This course focused on the foundations of artificial intelligence, including responsible AI use, prompt writing, and practical ways AI tools can support productivity and learning. It helped me improve how I interact with AI systems and gave me a better understanding of their real-world use.

Tools used: Coursera, Google Gemini

Java - Distinct Numbers

This Java lab required the user to enter numbers and then display only the distinct values that were entered. Although the concept seemed simple, the assignment challenged me to think carefully about logic, data handling, and how to avoid repeated values. It was a strong exercise in building my foundational Java skills.

Tools used: Java, IntelliJ IDEA
